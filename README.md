# Gerenciador de Tarefas (CLI) — Projeto Integrador

Sistema simples em Python (terminal) para cadastrar, listar, concluir, editar e remover tarefas, com persistência em arquivo JSON.

## Objetivo
Ajudar estudantes a organizar tarefas e prazos de forma prática, treinando fundamentos de programação e estruturação de sistemas.

## Funcionalidades
- Adicionar tarefa (título, descrição)
- Listar tarefas (pendentes/concluídas)
- Marcar como concluída
- Editar tarefa
- Remover tarefa
- Salvar e carregar automaticamente via `tarefas.json`

## Tecnologias
- Python 3
- Terminal/CLI
- JSON (persistência de dados)

## Como executar
1. Instale o Python 3.
2. Abra a pasta do projeto no VS Code.
3. No terminal, rode:

```bash
python gerenciador_tarefas.py
